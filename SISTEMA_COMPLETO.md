## 🚀 **Como Usar o Sistema**

### **1. Instalação**
```bash
# Clone o repositório
git clone [url-do-repositorio]

# Instale as dependências
npm install

# Configure o banco de dados
npm run db:push

# Execute o seed inicial
npm run db:seed

# Inicie o servidor
npm run dev
```
